# German M&A Market Analysis — 2025 Q1–Q3
M&amp;A Landscape in Germany: An Exploratory Data Analysis of Deal Activity and Valuation Multiples

### *Analyzing Germany’s M&A activity in 2025 using Python.*
This project explores **M&A transactions involving German companies** during **Q1–Q3 2025**, using deal-level data from **FactSet**.  
It uncovers trends in **industry activity, cross-border participation, and valuation multiples**.

## Key Insights
- **847** announced between Jan–Sep 2025; activity peaked in **Q1**.
- **Top sectors:** Technology 
- **Cross-border share:** 53.2% %, led by buyers from **U.S., Switzerland, and U.K.**
- **Highest EV/EBITDA multiples:** Healthcare  
- **Median disclosed deal value:** $68.9 million
